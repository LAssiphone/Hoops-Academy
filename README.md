# 🏀 Hoops Academy

Обучающая баскетбольная игра для детей 9–12 лет: учимся **двигаться без мяча**.

**4 урока:** Give-and-go · V-образный рывок · Бэкдор · Заслон.
Управление: левый джойстик — бег, правый — направление паса. Резкая смена направления даёт рывок.

Один самодостаточный HTML-файл, без зависимостей. Открой `index.html` — и играй.
